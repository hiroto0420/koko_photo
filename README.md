# koko_photo

## サービス概要
地図アプリと写真投稿型SNSを掛け合わせたサービスです。
写真を投稿する際に地図上で場所を指定し、どこで撮った写真なのかわかるように共有するサービスです。
投稿された写真を見たユーザーが自分も「行ってみたい」、「食べてみたい」と感じたときにすぐに地図上で確認することができます。

## このサービスへの思い・作りたい理由
私は休日にどこかに行きたいけど、どこに行こうと思った際にSNSを使って情報収集することが多々あります。その際、気になる投稿があった場合は地図アプリを立ち上げ場所を調べ、遠かった場合は再度SNSで情報収集することになります。地図アプリとSNSを同じサービス内で完結することでこのような作業の煩わしさを軽減できると考えました。
また、SNSに投稿する人は楽しかったことや印象に残っている場面を写真に残し、共有したいと感じていることでしょう。そして、その写真を見たユーザーが自分も「行ってみたい」、「食べてみたい」と感じ、実際に訪れたときに投稿したユーザーと投稿を見たユーザーの間で新たな繋がりが生まれるのではないかと考えました。
本サービスがこのようなユーザー同士のコミュニケーションの場となることを願っています。

## ユーザー層について
- SNSを使って観光地や飲食店の情報収集をしたい人
- 景色や食べ物などの思い出を共有したい人
- 自分の経験や体験を地図上に思い出として残したい人

## サービスの利用イメージ
### SNS機能
写真のアップロード及び位置情報の登録を必須とした投稿機能を実装します。

### 地図機能
投稿のある位置情報を地図上で表示します。
投稿の詳細画面と地図画面を双方向で遷移できるようにし、「投稿を見たユーザーが地図上で場所を確認できる」及び「地図の表示から近くの投稿を確認できる」を実現します。

## サービスの差別化ポイント・推しポイント
### 既存アプリとの比較
既存アプリとして地図アプリ（Googleマップ等）やSNSアプリ（Instagram等）など独立して存在します。
本サービスは上記のようなアプリを掛け合わしたものとなります。

### 想定する利用シーン1（投稿を見たユーザーが位置情報を確認）
SNSの写真を見たユーザーが自分も「行ってみたい」、「食べてみたい」と感じたときにどこで撮った写真なのか調べる手間を省きます。
また、景色などの写真については写真を見たユーザーが自分も「行ってみたい」と感じたとしても、どこで撮った写真かわからないことがありますが、本サービスではそのような問題を解消できます。

### 想定する利用シーン2（地図上から近くで投稿された写真を確認）
例として、旅先で昼食に困ったときに地図から近くの投稿を確認し、おいしそうな料理の出るお店を探すことができます。

## 機能候補
### MVP
- 新規登録・ログイン機能
- 位置情報を指定した写真投稿機能
- 投稿のある位置情報を地図上で表示

- コメント機能
- 投稿の公開範囲設定
- レスポンシブデザイン
- 投稿及びユーザーの検索・絞り込み機能
